---
title: "Listar Jobs"
description: "Lista todos os jobs da conta autenticada com paginação"
api: "GET /jobs"
---

Retorna uma lista paginada dos jobs da conta autenticada, ordenados por data de criação decrescente (mais recentes primeiro).

## Autenticação

Requer `access_token` (cookie) ou `Authorization: Bearer <api_key>`.

## Query Parameters

<ParamField query="status" type="string">
  Filtra por status. Valores válidos:
  - `pending_approval`
  - `queued`
  - `processing`
  - `completed`
  - `failed`
  - `cancelled`
</ParamField>

<ParamField query="page" type="integer" default="1">
  Número da página (começa em 1).
</ParamField>

<ParamField query="per_page" type="integer" default="20">
  Itens por página. Máximo 100.
</ParamField>

## Resposta

<ResponseField name="data" type="array">
  Lista de jobs (formato resumido, sem `result_file_url` ou detalhes de erro).

  <Expandable title="Campos de cada job">
    <ResponseField name="id" type="string">UUID do job.</ResponseField>
    <ResponseField name="status" type="string">Status atual.</ResponseField>
    <ResponseField name="modules" type="array">Módulos executados.</ResponseField>
    <ResponseField name="total_numbers" type="integer">Total de números.</ResponseField>
    <ResponseField name="processed_count" type="integer">Números processados.</ResponseField>
    <ResponseField name="valid_count" type="integer">Números válidos.</ResponseField>
    <ResponseField name="country_code" type="string">Código do país.</ResponseField>
    <ResponseField name="estimated_cost" type="number">Custo estimado em créditos.</ResponseField>
    <ResponseField name="actual_cost" type="number">Custo real (ou `null` se não concluído).</ResponseField>
    <ResponseField name="created_at" type="string">Data de criação (ISO 8601).</ResponseField>
    <ResponseField name="completed_at" type="string">Data de conclusão (ou `null`).</ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="pagination" type="object">
  Informações de paginação.

  <Expandable title="pagination">
    <ResponseField name="page" type="integer">Página atual.</ResponseField>
    <ResponseField name="per_page" type="integer">Itens por página.</ResponseField>
    <ResponseField name="total" type="integer">Total de registros.</ResponseField>
    <ResponseField name="pages" type="integer">Total de páginas.</ResponseField>
    <ResponseField name="has_next" type="boolean">Há próxima página?</ResponseField>
    <ResponseField name="has_prev" type="boolean">Há página anterior?</ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>

```bash cURL — todos os jobs
curl https://foneclear-web-production.up.railway.app/api/v1/jobs \
  -b cookies.txt
```

```bash cURL — apenas jobs concluídos
curl "https://foneclear-web-production.up.railway.app/api/v1/jobs?status=completed&page=1&per_page=10" \
  -b cookies.txt
```

```python Python
import httpx

r = httpx.get(
    "https://foneclear-web-production.up.railway.app/api/v1/jobs",
    headers={"Authorization": "Bearer vn_sk_live_a1b2c3d4e5f67890"},
    params={"status": "completed", "per_page": 10},
)
data = r.json()
for job in data["data"]:
    print(job["id"], job["status"], job["total_numbers"])
```
