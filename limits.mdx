| Limite | Valor |
|--------|-------|
| Tamanho máximo por upload | **50 MB** |
| Formatos aceitos | CSV, XLSX |
| Codificação CSV | UTF-8 (recomendado) ou Latin-1 |

**Custo em créditos por módulo:**

| Módulo | Créditos por número |
|--------|---------------------|

<Info>

</Info>

## Fluxo de aprovação

Por segurança contra abuso, todos os jobs passam por uma etapa de aprovação administrativa:

```
pending_approval  →  (admin aprova)  →  queued  →  processing  →  completed
                  →  (admin rejeita) →  cancelled
```

| Status | Descrição |
|--------|-----------|
| `pending_approval` | Aguardando revisão do administrador |
| `queued` | Aprovado, na fila de processamento |
| `processing` | Em processamento pelo worker Celery |
| `completed` | Concluído com sucesso |
| `failed` | Falhou durante o processamento |
| `cancelled` | Cancelado (pelo usuário ou admin) |

## Regras do módulo PRESENCA_HUMANA

Este módulo realiza chamadas telefônicas reais e está sujeito a restrições operacionais obrigatórias:

### Janela de horário

| Regra | Valor |
|-------|-------|
| Horário permitido | 08h00 às 20h00 |
| Fuso horário | America/Sao_Paulo (BRT/BRST) |
| Comportamento fora do horário | Número marcado como `amd_result: "pending"` sem custo |

Números marcados como `AMD_PENDENTE` **não geram custo** e são disponibilizados no relatório indicando que a verificação precisa ser reprocessada dentro da janela.

### Rate limits de voz

| Limite | Valor |
|--------|-------|
| Chamadas por segundo por número de origem | 1 CPS |
| Chamadas por hora por conta (fase inicial) | 200 |
| Pool mínimo de números de origem | 5 números (rotação round-robin) |

### Circuit breaker automático

O sistema pausa automaticamente um número de origem se:
- **ASR (Answer Seizure Ratio)** cair abaixo de 40%
- **Taxa de reclamação** ultrapassar 0,1%

### Lista DNC

Números registrados no **ANATEL Não Perturbe** (`naoperturbe.com.br`) são automaticamente excluídos das chamadas e categorizados como `DNC_SKIP` no resultado.

## Segurança

| Controle | Configuração |
|----------|-------------|
| Lockout de login | 5 tentativas falhas → bloqueio de 15 min |
| Senha mínima | 8 caracteres |

| HTTPS | Obrigatório (TLS 1.2+) |
| Isolamento entre contas | Row Level Security (PostgreSQL) |
